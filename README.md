# DE-Movielens_ml-20m

Coding Challenge:

Dataset - https://files.grouplens.org/datasets/movielens/ml-20m.zip

•             Understand the dataset and create a data structure

•             Develop a data pipeline to ingest the data into a database of choice and store data on-demand and
	      relay the status about data ingestion.

•             Use appropriate data warehouse modelling to scale and accommodate 100 million records

•             Develop an API service to:

		o             Retrieve movies and details based on the genres, year range and rating

		o             swagger documentation
